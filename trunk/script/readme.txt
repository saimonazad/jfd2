jFD2用スクリプトファイル集

※これは何？
jFD2はGroovyで書かれたスクリプトを実行する機能があります。
このスクリプトファイル集は、jFD2の作者である私が作成し、普段から使ってるスクリプトから比較的有用かつ無難な物を選んだ物です。

※使い方
このアーカイブを解凍し、同梱の拡張子が.groovyのファイルをjFD2のスクリプトファイル用ディレクトリにコピーしてください。
スクリプトファイル用ディレクトリは、jFD2を起動して、メニューバーの「オプション」→「スクリプト管理」を選択すると開かれるディレクトリになります（設定画面で変更可能です）。
コピーが出来たら、jFD2でOキーを押し（またはメニューバーの「スクリプト」→「スクリプトを選択して実行」）、コンボボックスから実行したスクリプトを選択して「OK」を押してください。
選択されたスクリプトが起動されます。
なお、最初の一回はスクリプトエンジンの起動のため、少々時間がかかります。

※各スクリプトの解説
・m3uファイル作成.groovy
Windows Media Center等のプレイリストファイルである、m3uファイルを作るためのスクリプトです。
MP3ファイルをマークしてこのスクリプトを起動するとファイル名を聞かれるので、入力するとその名前のm3uファイルを作成します。
jFD2はこのm3uファイルを解釈して、連続再生することも可能です。

・URL展開コピー.groovy
ダウンロードソフトのIrvine等で使用可能だった、連番のURLを展開して、カレントディレクトリにコピーするスクリプトです。
起動するとURLを訪ねられるので、そこに連番のURL（例：http://www.xxx.com/[1-100].jpg）を入力すると、展開してダウンロードします。

・ダウンロード.groovy
URLを入力すると、そのHTMLからリンクされた画像、動画、音楽ファイルをカレントディレクトリにダウンロードするスクリプトです。
Aタグでリンクされたファイルのみ取得します。
一部の画像掲示板で動作確認しています。

・ディレクトリツリー作成.groovy
・ディレクトリツリー作成（ディレクトリのみ）.groovy
起動するとカレントディレクトリ以下のファイルツリーをテキスト形式で表示するスクリプトです。

・ハッシュ値表示.groovy
選択されたファイルのハッシュ値を計算し、表示するスクリプトです。

・ファイルリスト作成.groovy
カレントディレクトリ以下、全てのファイルの一覧を表示するスクリプトです。
DVD-Rのファイルの整理用に作成を依頼された物です。

・ファイル振り分け.groovy
・ファイル振り分け（コピー）.groovy
マークされたファイルを、指定されたディレクトリにファイル名でサブディレクトリを切って振り分けるスクリプトです。
例えば、
１，a[xxx].txt
２，b[xxx].jpg
３，c[yyy].mpg
４，d[zzz].mp3
という4つのファイルをマークしてこのスクリプトを起動し、C:\testというディレクトリを入力すると、
１と２は「C:\test\xxx」、３は「C:\test\yyy」、４は「C:\test\zzz」というディレクトリに移動します（二つめのスクリプトは移動ではなくコピー）。

・個別圧縮.groovy
マークされたディレクトリを、それぞれ「（ディレクトリ名称）.zip」というファイル名で圧縮します。

・画像振り分け.groovy
画像を確認しながらサブディレクトリに振り分けます。
起動するとダイアログが表示されるので、振り分けたいサブディレクトリ名を入力してください。
ダイアログにその名前のチェックボックスが追加されます。
OKボタンを押すか何も入力せずにエンターを押すと、チェックされたディレクトリに表示中の画像をコピーし、次のファイルの振り分けに移ります。
沢山の人が来たパーティで撮った写真を、写っている人ごとに振り分ける友人のために作成したスクリプトです。
